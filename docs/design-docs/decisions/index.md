# Decision and contract-gate register

Status: canonical as of 2026-07-23.

## Wave 0 dispositions

- **Accepted:** `DEC-001` through `DEC-043`. These product and architecture
  decisions are canonical; acceptance does not claim that gated provider behavior
  exists.
- **Amended during integration:** `DEC-029` is now installed rather than proposed;
  `DEC-030` uses only commands that exist today; `DEC-031` uses a standard-library
  JSON manifest in Wave 0 and adds executable import enforcement in Wave 1;
  `DEC-039` selects manual one-agent-per-worktree dispatch until the Symphony
  spike passes; `DEC-042` defers the exact TanStack Query pin to the scaffold ADR.
- **Deferred product decisions:** `DW-FUT-101` through `DW-FUT-103`,
  `DW-FUT-201` through `DW-FUT-207`, and `DW-FUT-301` are accepted briefs, not v1
  implementation scope.
- **Unresolved external contracts:** all 44 `SPIKE-*` rows below remain open.
  Their deterministic fallback is the shipped behavior until accepted evidence is
  added to the source ledger. None is represented as ready.
