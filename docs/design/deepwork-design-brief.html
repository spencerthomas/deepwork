Deep Work · Design brief · v0.1 · 21 Jul 2026

An ecosystem-native workspace for agents that do real work

Deep Work is the open-source Codex / Cowork-style client for the LangChain platform: sign in with your LangSmith org, dispatch tasks to deep agents, steer them live, approve their actions, and manage a fleet of agents — from web, desktop, and mobile. The UI is a deliberate near-carbon copy of docs.langchain.com, so it feels like a first-party LangChain surface. Every value marked confirmed was extracted from LangChain's own sources; adapted marks our substitutions.

1 · Design north star

“Feels like LangChain shipped it.” This is a contribution to the LangChain ecosystem, not a third-party brand exercise. Calm, documentation-grade surfaces; dense but never cramped; the accent blue reserved for what's active or actionable. The product should read as an operations room for work being done by agents — lists, threads, diffs, approvals — not a chat toy.

  • Three product pillars to design for: a task inbox (dispatch → watch → steer → review), an approvals surface (what agents need from you), and a fleet manager (create, configure, schedule agents).
  • Calm by default, alive when running: state is shown with tint pills and small motion (a breathing status dot), never spinners everywhere.
  • The trace is truth: every run links to its LangSmith trace; the UI never hides what the agent actually did.

OAuth with LangSmith (device flow on desktop), workspace picker, API-key fallback. Design Q: how to make “your org, your data” legible — this is the wrapper's trust story.

9 · Do & don't

Do

  • Copy the docs shell dimensions and interaction details exactly — that's the brief.
  • Use blue-tinted neutrals (the gray ramp is derived from #161F34).
  • Reserve #006DDD / #7FC8FF for active, actionable, running.
  • Design light and dark together; dark surfaces step #030710 → #0D1322 → #161F34.
  • Show provenance everywhere: trace links, sandbox IDs, PR links.

Don't

  • Ship the LangChain wordmark/logo or the name “LangChain” in the brand (registered trademark; this is an unaffiliated contribution until they adopt it).
  • Embed TWK Lausanne or Aeonik Mono font files (commercial licenses).
  • Use pure black; the darkest ink is #030710 / #0A0B0B.
  • Invent a second accent hue — semantic colors come from the extended palette.
  • Bury agent actions behind “magic” — every mutation is inspectable.
