# TypeScript proof-consumer drafts

Status: **non-canonical review history**
Created: 2026-07-23

This package preserves unreviewed TypeScript proof-consumer planning drafts from
an external planning bundle. These records are not active ExecPlans, are not
indexed, and do not authorize dispatch or implementation. A coordinator may
promote a draft only after independent review, complete active-ExecPlan metadata,
resolvable dependency evidence, and index registration.

The API-to-SDK draft names the completed `local:DW-M1-API-001` scaffold record
as its resolvable API dependency. It remains blocked until a separately accepted
successor supplies the final-wire API evidence it requires; this archive does not
invent or register that successor.
