create_graph requires an injected BaseChatModel and accepts optional LangChain
tools plus an optional caller-owned checkpointer. Its public flow is plan,
LangGraph approval interrupt, Deep Agents execution or rejection, and terminal
output. The default checkpointer is in-memory only.

Importing the package performs no network call, reads no environment variable,
and does not create a model, graph, provider client, deployment, application
session, or durable persistence. runtime_capabilities reports only
local-runtime availability and reports external providers as unavailable.
